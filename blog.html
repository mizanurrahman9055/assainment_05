Question-1: What are the different ways to select an element in the DOM?

Answer:The different ways to select an element in the DOM are:getElementById(),getElementsByClassName(),querySelector() and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML: Gets or sets the HTML content inside an element (includes HTML tags).
innerText: Gets or sets the visible text inside an element (ignores hidden elements and extra spaces).
textContent: Gets or sets all text inside an element (includes hidden elements but ignores HTML tags).

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique where a parent element handles events for its child elements using event bubbling, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is the process where an event starts from the target element and propagates up through elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: Create: Use document.createElement("tagName").
Add: Use appendChild() or append().
Remove: Use element.remove().
